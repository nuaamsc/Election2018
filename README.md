# Election2018
南航微软学生俱乐部第18届主席团成员选举
## 选举流程
1. 邀请俱乐部部长团中优秀的成员填写[申请表]()
2. 对填写申请表的同学进行一对一面谈，确定其意见
3. 申请主席团的同学进行答辩，答辩评委由 现任主席团成员，指导老师，特聘顾问担任。
